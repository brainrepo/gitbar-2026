## Descrizione

Puntata speciale fuori programmazione per annunciare una notizia bomba: Microsoft ha acquisito npm attraverso GitHub. Parliamo di come questo cambi il panorama dello sviluppo JavaScript e di cosa possiamo aspettarci dal futuro. GitHub promette che il public registry rimarrà com'è, ma con l'integrazione della potenza di Azure e dei GitHub Security Labs. Microsoft sta completando il suo poker d'assi nell'ecosistema JavaScript: TypeScript, Visual Studio Code, GitHub e ora npm. Il pacchetto è completo e la strategia è chiara: conquistare la community più numerosa del mondo dello sviluppo.

## Takeaway

- npm rimane pubblico e free, ma si integrerà con l'infrastruttura Azure di Microsoft
- Promessa di coordinamento tra pull request e pubblicazione nel registry con workflow più semplici
- Integrazione con GitHub Security Labs per maggiore sicurezza dei pacchetti
- Possibilità di collegamento con GitHub Sponsors per supportare i progetti open source
- Microsoft completa il dominio sull'ecosistema JavaScript: TypeScript, VS Code, GitHub e ora npm

## Bold Opinion

- Microsoft sta facendo all-in sui tool per developer e la strategia funziona alla grande
- GitLab inizia a farsi sentire come concorrente, ma GitHub resta una spanna sopra per numeri e adozione
- La community di JavaScript è ormai nelle mani di Microsoft, per il meglio o per il peggio
